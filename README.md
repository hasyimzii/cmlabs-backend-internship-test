# CMLABS Backend Internship Test
FizzBuzz algorithm code for pre-assessment test

## How to install
(Must've installed: Go Programming Language)

1. Clone this repository
```sh
git clone https://github.com/hasyimzii/cmlabs-backend-internship-test.git
```
2. Open repository in terminal
```sh
cd cmlabs-backend-internship-test
```
3. Build Go app
```go
go build
```
4. Excecute binary file
```sh
./cmlabs-backend-internship-test
```  
5. Now the code is running
